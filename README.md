# JustStreamIt — Frontend

Interface web **HTML / CSS / JavaScript** pour le projet OpenClassrooms : affichage des films via l’**API OCMovies** (locale).

## Prérequis

- API **OCMovies** lancée sur **`http://localhost:8000`** (endpoint utilisé : `/api/v1/`).

## Lancer

Ouvrir **`index.html`** dans le navigateur, ou servir le dossier avec un serveur HTTP local si besoin.

## Fichiers

- `index.html` — structure de la page  
- `style.css` — styles  
- `script.js` — appels `fetch` à l’API et logique d’affichage  
- `images/` — logos et visuels statiques
